# Gestor de Libros - Serialización Java

## Descripción
Aplicación en Java que demuestra el proceso de serialización de objetos. Permite guardar una lista de libros en un archivo binario y recuperarla posteriormente para visualizar los datos.

## Funcionalidades
- Crear objetos de tipo Libro (título, autor, año)
- Guardar colecciones de libros en archivos binarios (serialización)
- Leer y recuperar libros desde archivos binarios (deserialización)
- Mostrar los libros recuperados por pantalla

## Tecnologías
- Java
- Serialización (interfaz Serializable)
- ObjectOutputStream / ObjectInputStream
- FileOutputStream / FileInputStream
- ArrayList
- Manejo de excepciones IOException y ClassNotFoundException
